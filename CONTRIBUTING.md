## ğŸ”§ Code Contributions

### Setup
1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/qr-data-transfer.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test thoroughly on multiple devices
6. Commit: `git commit -am "Add feature: description"`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

### Guidelines

- Keep code clean and well-commented
- Maintain single-file HTML structure for simplicity
- Test on both desktop and mobile browsers
- Ensure HTTPS requirements are documented
- Update README if adding features
- Follow existing code style

### Testing Checklist

- [ ] Sender works on desktop
- [ ] Sender works on mobile
- [ ] Receiver works on mobile with HTTPS
- [ ] File transfer works correctly
- [ ] Text transfer works correctly
- [ ] Progress bars update properly
- [ ] Error handling works
- [ ] UI is responsive
